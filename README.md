# MongoDB-Basic Practice Task
# 📘 MongoDB CRUD – Student Database

## 📌 Objective
To learn and perform CRUD operations in MongoDB using a student database.

---

## 🗂️ Database & Collection
- Database: `studentDB`
- Collection: `students`

---

## 🔹 Operations Performed

### ✔ Create
- Created database and collection
- Inserted single and multiple student records

### ✔ Read
- Retrieved all documents
- Filtered students enrolled in "MERN Stack"

### ✔ Update
- Updated status of a student
- Updated multiple student records

### ✔ Delete
- Deleted one student document
- Deleted all documents for practice

---

## 🔹 Query Operators Used
- `$gt`, `$lt` → Age filtering  
- `$in` → Multiple course filtering  
- `$and`, `$or` → Combined conditions  
- `$exists` → Field checking  

---

## 💡 Use Case
This database simulates a **Student Management System**, where student details like name, course, and status are managed efficiently.

---

## 🎯 Outcome
- Learned CRUD operations  
- Practiced query operators  
- Improved understanding of MongoDB  

---

## 🏁 Conclusion
This task helped in understanding how to manage student data using MongoDB and perform operations efficiently.
